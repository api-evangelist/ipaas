About Reciprocity Providers

This is a project site to track on the what I call reciprocity providers..

I host individual research projects and sections of API Evangelist as their own Github repositories. This way projects can evolve on their, independint of the overall site.

Reciprocity providers represents an important trend in the evolution of APIs in an effort deal with migrating data between cloud providers.

If you have any questions about this research, contact me at info@apievangelist.com.
