The most exciting new thing to come out in this latest release is the new Dropbox connector.You will find it in the Toolbox in the Warewolf Studio.It looks like this once on the design surface: What the Dropbox connector highlights is just how easy it is to create your own Warewolf tools.We’ve documented it in a simple step-by-step guide “Getting started with tools and data connectors” in the knowledge base.Here you can learn how to easily create both basic and more complex Warewolf tools and data connectors.
