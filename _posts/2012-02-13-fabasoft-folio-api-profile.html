Fabasoft Folio Cloud is cloud based software that offers enterprises a number of business applications within a single cloud computing environment.Developers can use Folio Cloud as a platform on which to develop their own apps.APIs are available that let users access their business data using open standards such as WebDAV, CalDAV and CMIS.CalDAV - Synchronize a calendar in Folio Cloud with a mobile device.WebDAV - Synchronize content from Folio Cloud to a local computer and access documents by double-click.
