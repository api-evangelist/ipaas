Share this article: Today we’ll tackle how to utilize CloverETL with Facebook.Facebook offers Graph API to allow applications to access social graphs.This blog post describes how to use Graph API within CloverETL.We’ll use App Access Token, as it doesn’t require user interaction during the generation process and does not expire.Check out the documentation to see the differences between available tokens and their limitations.
